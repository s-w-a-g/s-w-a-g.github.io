Generate your static website with Sw:ag!

Sw:ag is an easy to use static website generator.

Assets organization

Source assets should be located in a swag directory as follows:

📁 User directory to process
│
├──📁 swag
│   ├──📁 data
│   └──📁 pages
…

pages directory

The structure of the pages directory matches very closely the static_website organization. 
📁 pages
│
├──❌ .hidden_file
├──📁 blog
│   ├──📁 post
│   │   └──🔃 article.html.twig
│   └──❗️index.html.twig
├──⭕️base.html.twig
├──⭕️header.html.twig
├──❗️contact.html.twig
├──❗️index.html.twig
├──📁 style
│   ├──📁 css
│   │   └──❕main.css
│   └──📁 js
│       └──❕app.js
…
➡︎ 
📁 static_website
│
├──📁 blog
│   ├──📁 post

data directory

Sw:ag generates a structured data based on the data directory tree.

The following organization 
📁 swag
│
├──📁 data
│   ├──📁 posts
│   │  ├──📄 post_1.md
│   │  ├──…
│   │  └──📄 post_n.md
│   └──📄 site.yml
…
➡︎
will generate the following data: 
data:
    site:
        url: myurl.com
        title: My generated website
        …
    posts:
        post_1:
            author:  Jane Doe
            date:    2016-12-01
            content: An awesome post… chit-chat
        …
        post_n:
            author:  Max Cool
            date:    2016-06-22
            content: Another article… chit-chat
